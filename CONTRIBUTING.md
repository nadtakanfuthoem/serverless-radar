# Contributing to Serverless Radar

Thanks for your interest in contributing! Here's how to get started.

## How to Contribute

1. **Fork** the repository
2. **Create a branch** for your change
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes** and commit with a clear message
   ```bash
   git commit -m "feat: add support for X"
   ```
4. **Push** your branch and open a **Pull Request**

## Ideas for Contributions

- Add support for filtering by additional AWS service categories
- Add JSON or CSV output format
- Add a scheduled/cron mode to run automatically
- Add tests

## Code Style

- Use ES modules (`import`/`export`)
- Keep functions small and focused
- Add comments for non-obvious logic

## Reporting Issues

Open an issue with a clear description of the bug or feature request.
